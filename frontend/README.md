# 🎬 SubsTranslator Frontend

ממשק משתמש מודרני ויפה למתרגם כתוביות חכם עם AI.

## ✨ תכונות

- 📁 **העלאת קבצים** - תמיכה בפורמטים מרובים (MP4, MKV, MOV, WebM, AVI, MP3, WAV)
- 📺 **YouTube** - הורדה ועיבוד ישיר מ-YouTube
- 🌍 **תרגום** - תמיכה ב-10+ שפות שונות
- 🎯 **ממשק נגיש** - עיצוב RTL לעברית
- 📱 **רספונסיבי** - מתאים לכל גדלי מסכים

## 🚀 התחלה מהירה

### דרישות מוקדמות
- Node.js 16+ 
- npm או yarn
- שרת Backend פעיל (ראה ../backend)

### התקנה

```bash
# התקנת תלויות
npm install

# הפעלה במצב פיתוח
npm start
```

האפליקציה תיפתח בדפדפן בכתובת: [http://localhost:3000](http://localhost:3000)

### הגדרת Backend

ודא שהשרת Backend פועל על http://localhost:8081

```bash
cd ../backend
python app.py
```

## 🛠 סקריפטים זמינים

### `npm start`
הפעלה במצב פיתוח עם Hot Reload

### `npm run build`
בניית גרסת ייצור מינוס לתיקיית `build`

### `npm test`
הפעלת בדיקות האפליקציה
